# 💳 Credit Card Fraud Detection System

## 📌 Description
Developed a machine learning model to detect fraudulent credit card transactions using SMOTE for imbalanced data. Implemented risk scoring and alert system to identify high-risk transactions.

## ⚙️ Technologies
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn
- Imbalanced-learn

## 🚀 Features
- Fraud Detection Model
- SMOTE for imbalanced data
- Risk Scoring System
- Alert Classification (High/Medium/Low)
- Data Visualization

## 📊 Dataset
Kaggle Credit Card Fraud Detection Dataset
